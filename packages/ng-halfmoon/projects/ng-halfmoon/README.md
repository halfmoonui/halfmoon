# NgHalfmoon

## Installation
You can install `ng-halfmoon` via npm
```bash
npm install ng-halfmoon
```

## Setup
Simply import the `NgHalfmoonModule` into your Application

```typescript
import {NgHalfmoonModule} from "ng-halfmoon";

@NgModule({
  imports: [
    NgHalfmoonModule
  ],
})
export class AppModule {
}
```
